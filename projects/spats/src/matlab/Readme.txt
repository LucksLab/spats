MATLAB Code for

"Modeling and automation of sequencing-based characterization of RNA structure"
Sharon Aviran, Cole Trapnell, Julius B. Lucks, Stefanie A. Mortimer, Shujun Luo, Gary P. Schroth, Jennifer A. Doudna, Adam P. Arkin, and Lior Pachter
PNAS, 2011

Extensive comments are found inside the files.

MLE.m - Standalone program for calculating MLE parameters for one molecule.
MLE_one_molecule.m - Function form of MLE.m
barcodes_statistics.m - Driver code for MLE_one_molecule.m over a series of barcodes.
bootstrap_error_bars.m - Computing bootstrap error bars for one molecule.

The rest of the files are functions called within those programs.
